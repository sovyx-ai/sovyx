# Sovyx

**The AI Mind That Remembers You.**

Sovyx is the Sovereign Minds Engine — a persistent AI companion with real
memory, a cognitive loop, and a brain graph. It ships as a Python library,
a CLI daemon (`sovyx`), and a React dashboard.
